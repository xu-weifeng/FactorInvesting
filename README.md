# Multi-Factor Investing

Construction and evaluation of equity factor portfolios using Python.

## Project Overview

This project investigates whether classical equity factors generate persistent excess returns and whether combining multiple factors improves portfolio performance.

The study focuses on three representative equity factors:

- Size
- Value
- Momentum

The project includes the complete research workflow, including data collection, factor construction, portfolio formation, backtesting, performance evaluation, visualization, and academic reporting.

---

## Research Question

Can classical equity factors generate persistent excess returns, and does combining multiple factors improve portfolio performance?
